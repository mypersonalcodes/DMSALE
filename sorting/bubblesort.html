Bubble Sort Explanation

What is Bubble Sort?

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted.

How does Bubble Sort work?

The algorithm works as follows:

  1. Compare the first two elements. If the first is greater than the second, swap them.
  2. Move on to the next pair of elements and repeat step 1.
  3. Repeat steps 1 and 2 for all pairs of elements, going from the beginning to the end of the list.
  4. Repeat the entire process until the list is sorted.

Example

Let's sort the list [5, 2, 9, 1, 7] using Bubble Sort.

Pass List
1 [2, 5, 1, 7, 9]
2 [2, 1, 5, 7, 9]
3 [1, 2, 5, 7, 9]
4 [1, 2, 5, 7, 9]

Pseudocode

    def bubble_sort(arr):
    n = len(arr)

    # Traverse through all array elements
    for i in range(n):
        # Last i elements are already in place
        for j in range(0, n - i - 1):
            # Traverse the array from 0 to n-i-1
            # Swap if the element found is greater
            # than the next element
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

    return arr

Complexity

The time complexity of Bubble Sort is O(n^2), where n is the size of the input list. This is because the algorithm has to make n passes through the list, and in each pass, it has to compare n-1 pairs of elements.
